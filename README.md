# Loans for good
O projeto consiste em uma API para solicitação de empréstimo e um frontend com formulário dinâmico.

## Stack backend
- Python 3.8
- Django 3.2
- DRF
- Docker
- Celery 3.1
- RabbitMQ
- Postgres

## Stack frontend
- HTML
- JS
- CSS

## Build do projeto
### O build do projeto irá também criar o usuário admin.
```
make build
```
